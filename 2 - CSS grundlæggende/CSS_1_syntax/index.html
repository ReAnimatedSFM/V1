Om CSS

Hvad er en selector med egne ord?

En selector vælger HTML elementet som man ønsker at stilisere.

Hvad er en property med egne ord?

Property bruges til at definere stiliseringen, fx, font-size som bestemmer hvor stor skriftstørrelsen skal være.

Hvad er en value med egne ord?

Value er værdien som du giver til din property, fx, blue(color), 18px(size), url(billede)(), etc...

Oversigt over simple selectors
Selector: Eksempel: Forklaring:
Element p Vælger alle elementer af denne type, f.eks. alle paragraphs
Id #someId Id vælger ét element med følgende id og stilisere det.
Class someClass Vælger alle elementer med klassen
Universal * Vælger ALLE elementer
Grouping td, th, tr, table Vælger alle elementer af angivne typer
