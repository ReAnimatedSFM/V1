HTML 1

HTML står for HyerText Markup Language.

HTML blev opfundet af Tim Bernes-Lee i 1991.

HTML5 er den nyeste version af HTML.

Indeholdet i body er det synlige side indhold.

head kan indholde title, meta, style, link og SKAL indholde title


index.html, default.html og default.htm, fungere som en 'standard side', så nå en browser beder om et URL. Hvis index.html(etc..) ikke existere, kan web serveren konfigureres til at give en automatisk-frembringet liste af filerne inde i directorien istedet.
